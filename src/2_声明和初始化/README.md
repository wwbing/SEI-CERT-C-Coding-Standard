速记：
	1：使用模板参数包template <typename... Ts>替代C风格的...
	2：声明或定义时注意保留运算符，其一般特点（_Value, __value, 以及全局命名空间下的_value）
	3：不要用const或者volatile修饰引用变量
	4：编译器在能处理成声明时会优先处理为声明（要明确写法）
	5：如果你在某个作用域里重载了分配函数（new, new[]），就必须在同一个作用域里也重载对应的释放函数（delete, delete[]）；反过来也一样。 
	6：避免静态对象初始化期间的循环；静态对象的初始化时机很特殊，如果初始化过程中又反过来依赖自己，或者依赖一个还没准备好的静态对象，就会出问题。
